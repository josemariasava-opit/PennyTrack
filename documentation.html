PennyTrack Documentation

Everything you need to know about using and contributing to PennyTrack.

About PennyTrack

PennyTrack is a personal expense management application designed to help users track their income and expenses effectively. With an intuitive user interface, PennyTrack enables users to monitor their financial health and make informed decisions about their spending habits.

Getting Started

To get started with PennyTrack, simply navigate to the Tracker page and begin adding your transactions. You can categorize your income and expenses, view real-time analytics, and export your data for further analysis.

Setup Instructions

PennyTrack is a Progressive Web App (PWA) that works offline. To install it on your device:

  1. Visit the live demo link: Live Demo.
  2. On mobile devices, tap the "Add to Home Screen" option from your browser's menu.
  3. On desktop, you can bookmark the page for quick access.

Key Features

Expense Tracking

Add, edit, and delete expenses to keep an accurate record of your spending.

Income Tracking

Record various income sources to monitor your earnings.

Categorization

Organize transactions into categories for better analysis.

Financial Insights

Gain insights into your spending patterns to make informed financial decisions.

LocalStorage Support

All user data is stored in the browser's LocalStorage, ensuring persistence between sessions without requiring a backend.

Technologies Used

  • Frontend: HTML, CSS, JavaScript
  • Charts: Highcharts.js for dynamic data visualization
  • Storage: LocalStorage (no backend required)
  • Offline Support: Service Worker for Progressive Web App (PWA) capabilities

Frequently Asked Questions (FAQs)

Q: How do I export my transactions?
A: Navigate to the Tracker page and click the "Export CSV" button to download your transactions as a CSV file.

Q: Can I import transactions from a CSV file?
A: Yes! Use the "Import CSV" feature on the Tracker page to upload your transactions.

Q: Is my data secure?
A: Your data is stored locally in your browser using LocalStorage. It is not shared with any external servers.

Contributing Guidelines

We welcome contributions from the community! If you'd like to contribute to PennyTrack:

  1. Fork the repository on GitHub.
  2. Create a new branch for your feature or bug fix.
  3. Submit a pull request with a detailed description of your changes.

Please ensure your code follows the project's coding standards and includes appropriate documentation.
