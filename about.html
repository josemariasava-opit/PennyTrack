About PennyTrack

PennyTrack is an advanced Progressive Web App (PWA) designed for personal finance management. Developed as an assessment for my COMP-2002 web development course, taken at OPIT.

🔐 Local Storage

All financial data is securely stored in your browser using localStorage, ensuring complete privacy.

📱 PWA Features

Installable web app with offline support and native-like performance on any device.

📊 Dynamic Charts

Interactive visualizations powered by Highcharts for real-time financial charts.

🎓 Academic Project

Developed as part of my Computer Science degree assessment in web development.

📥 Data Import/Export

Easily backup or migrate your data using CSV file with Import/export functions.

🌍 Open Source

Available on GitHub for transparency, version control and hosting.

🛠️ Vanilla JS

Built with pure JavaScript without frameworks for optimal performance.

🌓 Theme System

Custom dark/light mode with localStorage persistence for user preferences.

📲 Responsive Design

Fully responsive layout that adapts to mobile, tablet, and desktop screens.
