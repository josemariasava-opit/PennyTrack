About PennyTrack

PennyTrack is an advanced Progressive Web App (PWA) designed for personal finance management. Developed as part of my COMP-2002 web development course at OPIT.

Key Features

🔒 Local Storage

All financial data is securely stored in your browser using localStorage, ensuring complete privacy.

📱 PWA Features

Installable web app with offline support and native-like performance on any device.

📊 Dynamic Charts

Interactive visualizations powered by Highcharts for real-time financial charts.

Data Management

📥 Import/Export CSV

Easily backup or migrate your data using CSV files. Transactions can be exported and imported seamlessly.

📅 Filtered Analytics

Filter transactions by year and month to analyze your financial data dynamically.

📝 Editable Transactions

Edit or delete transactions directly from the table for accurate record-keeping.

Technical Implementation

🛠️ Vanilla JavaScript

Built with pure JavaScript without frameworks for optimal performance and lightweight execution.

🌓 Dark/Light Mode

Custom dark/light mode with localStorage persistence for user preferences.

📲 Responsive Design

Fully responsive layout that adapts to mobile, tablet, and desktop screens.

Open Source & Academic Context

🎓 Academic Project

Developed as part of my Computer Science degree assessment in web development.

🌍 Open Source

Available on GitHub for transparency, version control, and hosting.

⚙️ Service Worker

Service worker registration ensures offline functionality and improved performance.
